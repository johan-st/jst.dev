# auth.jst.dev

A simple authentication server written in go. It is used to authenticate users for any website I host. It is also used to authenticate users for any ssh application I host.

## Features

- [ ] Single hardcoded user (me)
- [ ] Authentication
- [ ] Authorization
- [ ] User management
- [ ] Password reset
- [ ] Email verification
- [ ] 2FA
- [ ] OAuth
- [ ] SSO
- [ ] Audit log
- [ ] Admin panel
- [ ] API
- [ ] gRPC
- [ ] Monitoring
- [ ] Metrics
- [ ] Logging
- [ ] Tests
- [ ] CI/CD
- [ ] Documentation
